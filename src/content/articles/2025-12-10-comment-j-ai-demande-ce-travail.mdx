## Pourquoi ce journal existe
Je voulais un système :
- sans base de données
- où le contenu est en Markdown/MDX dans Git
- et où le site se rebuild à chaque commit.

## Comment je m’y suis pris pour demander ce travail
J’ai listé mes contraintes (SEO, images, pas de DB), puis j’ai raffiné :
- tags, catégories, listing
- recherche sans serveur
- lecture par séquences
- Decap CMS pour éditer en ligne
- déploiement Cloudflare Pages.

## Le résultat
Un starter Astro + MDX qui rend chaque `##` comme une **séquence** :  
le lecteur avance avec **→** et peut ouvrir le menu **☰**.
