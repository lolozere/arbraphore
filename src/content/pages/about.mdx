## Intention
Je veux un espace d’écriture **sans base de données**, versionné dans Git, et publié en site statique très rapide.

## Minimalisme
Pas d’entête. Une sidebar (logo, titre, recherche, menus).  
Sur mobile, un petit **chevron** permet de l’afficher.

## Médias
Une section peut être :
- du texte long (scroll autorisé)
- une image / vidéo / audio / embed (centré, sans scroll)

## Open
Tout est modifiable : styles, types de contenus, modèles RSS, etc.
